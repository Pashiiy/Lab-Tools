# Changelog

All notable changes to Lab Tools are documented here.
The format follows [Keep a Changelog](https://keepachangelog.com/) and [Semantic Versioning](https://semver.org/).

Conventional commit types for version bumps:
- `fix:` → patch
- `feat:` → minor
- `BREAKING CHANGE:` or `feat!:` → major

## [1.0.0] - 2026-06-11

### Features

- Unified Lab Tools shell with Endpoint Analysis, Colony Counter, and Gel Quantification
- Electron + Vite production builds for Windows (NSIS) and macOS (DMG)
- GitHub Actions release pipeline with automated publishing
